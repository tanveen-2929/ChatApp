# 💬 Chat Application (MERN + Socket.io)

A real-time chat application built using **MongoDB, Express, React, Node.js (MERN)** and **Socket.io** for instant messaging.

---

## 🚀 Live Demo
👉 https://chatapp-0xsv.onrender.com

---

## ✨ Features

- 🔐 User Authentication (Login / Register)
- 💬 Real-time messaging using Socket.io
- 👥 Multiple users chat support
- 📱 Responsive UI (Mobile + Desktop)
- ⚡ Fast and interactive UI
- 🟢 Live online status

---

## 🛠️ Tech Stack

**Frontend:**
- React.js
- Vite
- CSS

**Backend:**
- Node.js
- Express.js
- Socket.io

**Database:**
- MongoDB Atlas

**Authentication:**
- JWT (JSON Web Token)
- bcrypt.js
